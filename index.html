1. Introduction

Bienvenue dans ERP AI

ERP AI est une solution complète de gestion d'entreprise conçue spécifiquement pour les entreprises de construction québécoises. Ce système intègre tous les aspects de votre activité : projets, production, inventaire, ressources humaines et relation client.

🎯Fonctionnalités Clés

  • Gestion de projets avec 24 étapes de construction québécoises
  • Bons de travail avec suivi en temps réel
  • TimeTracker unifié pour pointage par opération
  • CRM intégré avec pipeline de vente
  • Inventaire avec support impérial/métrique
  • Assistant IA Claude pour analyses et recommandations
  • 61 postes de travail préconfigurés
  • Export PDF professionnel

💻Configuration Requise

  • Navigateur : Chrome, Firefox, Edge (dernières versions)
  • Résolution : Minimum 1280x720, recommandé 1920x1080
  • Connexion Internet : Requise pour l'Assistant IA
  • Système : Windows 10/11, macOS, Linux

📋Bons de Travail (BT)

Création d'un Bon de Travail

  1. Accès : Production → Nouveau BT
  2. En-tête du BT :
    • Numéro automatique (BT-2024-XXXXX)
    • Projet associé (obligatoire)
    • Client final
    • Date de création
    • Priorité (NORMAL, URGENT, CRITIQUE)
  3. Ajout des Opérations :
    Pour chaque opération : - Poste de travail (61 choix) - Description détaillée - Durée estimée (heures) - Employé assigné - Matériaux requis
  4. Validation et Envoi :
    • Brouillon : Sauvegarde sans validation
    • Validé : Prêt pour production
    • En cours : Démarré en atelier

Workflow des Bons de Travail

BROUILLON → VALIDÉ → EN_COURS → TERMINÉ ↓ ↓ ↓ ↓ [Création] [Approval] [Production] [Livraison]

5. Workflows Métier

🔄Workflow Projet Complet

1. COMMERCIAL CRM → Opportunité → Devis → Négociation ↓ 2. LANCEMENT Projet créé → Équipe assignée → Planning défini ↓ 3. PRODUCTION BT créés → Opérations planifiées → TimeTracker actif ↓ 4. SUIVI Progression temps réel → Ajustements → Contrôle qualité ↓ 5. LIVRAISON Tests finaux → Documentation → Facturation → Clôture

📋Workflow Bon de Travail

1. CRÉATION Projet sélectionné → Opérations définies → Ressources allouées ↓ 2. VALIDATION Vérification faisabilité → Matériaux disponibles → Approval ↓ 3. PRODUCTION Lancement atelier → Pointages TimeTracker → Suivi progression ↓ 4. CONTRÔLE Qualité vérifiée → Écarts analysés → Documentation ↓ 5. CLÔTURE BT terminé → Mise à jour projet → Facturation possible

🏭Workflow Production

1. PLANIFICATION Charge atelier → Capacité disponible → Ordonnancement ↓ 2. PRÉPARATION Matériaux réservés → Postes assignés → Gammes définies ↓ 3. EXÉCUTION Pointage début → Suivi temps réel → Alertes dépassement ↓ 4. AJUSTEMENTS Réaffectations → Priorisation → Optimisation flux ↓ 5. REPORTING KPIs production → Analyse écarts → Amélioration continue

8. Dépannage

❓Problèmes Fréquents

Connexion Impossible

Vérifiez

  • Identifiants corrects
  • Majuscules activées ?
  • Compte actif ?
  • Session expirée ?

Solutions

  • Réinitialiser mot de passe
  • Contacter administrateur
  • Vider cache navigateur
  • Essayer autre navigateur

Performance Lente

Causes possibles

  • Connexion Internet
  • Navigateur surchargé
  • Base données volumineuse
  • Trop d'utilisateurs

Optimisations

  • Fermer onglets inutiles
  • Redémarrer navigateur
  • Utiliser filtres données
  • Éviter heures de pointe

Erreurs d'Affichage

Vérifications

  • Zoom navigateur à 100%
  • JavaScript activé
  • Cookies autorisés
  • Mode privé désactivé

Corrections

  • F5 pour rafraîchir
  • CTRL+F5 (cache complet)
  • Désactiver extensions
  • Mettre à jour navigateur

🛠️Maintenance Utilisateur

Nettoyage Navigateur

Chrome/Edge : 1. Paramètres → Confidentialité 2. Effacer données navigation 3. Cocher : Cookies, Cache 4. Période : Toutes 5. Effacer les données

Optimisation Session

  • Déconnexion propre quotidienne
  • Pas plus de 10 onglets ERP
  • Éviter multiples fenêtres
  • Rafraîchir si inactive >30min

Bonnes Pratiques

Sauvegarde régulière

  • Avant modifications majeures

Documentation
