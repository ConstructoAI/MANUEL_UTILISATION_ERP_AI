📋 Bons de Travail (BT)

Création d'un Bon de Travail

1. Accès : Production → Nouveau BT

2. En-tête du BT :

  • Numéro automatique (BT-2024-XXXXX)
  • Projet associé (obligatoire)
  • Client final
  • Date de création
  • Priorité (NORMAL, URGENT, CRITIQUE)

3. Ajout des Opérations :

Pour chaque opération :
- Poste de travail (61 choix)
- Description détaillée
- Durée estimée (heures)
- Employé assigné
- Matériaux requis

4. Validation et Envoi :

  • Brouillon : Sauvegarde sans validation
  • Validé : Prêt pour production
  • En cours : Démarré en atelier

❓ Dépannage

Problèmes Fréquents

🔐 Connexion Impossible

Vérifiez :

  • Identifiants corrects
  • Majuscules activées ?
  • Compte actif ?
  • Session expirée ?

Solutions :

  • Réinitialiser mot de passe
  • Contacter administrateur
  • Vider cache navigateur
  • Essayer autre navigateur

🐌 Performance Lente

Causes possibles :

  • Connexion Internet
  • Navigateur surchargé
  • Base données volumineuse
  • Trop d'utilisateurs

Optimisations :

  • Fermer onglets inutiles
  • Redémarrer navigateur
  • Utiliser filtres données
  • Éviter heures de pointe
